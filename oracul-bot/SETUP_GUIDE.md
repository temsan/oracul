# 🔮 Руководство по настройке Oracul Bot

## Быстрый старт

### 1. Получение BOT_TOKEN

1. Откройте Telegram и найдите бота [@BotFather](https://t.me/BotFather)
2. Отправьте команду `/newbot`
3. Введите имя бота (например: "My Oracul Bot")
4. Введите username бота (например: "my_oracul_bot")
5. Скопируйте полученный токен

### 2. Настройка .env файла

Отредактируйте файл `oracul-bot/.env`:

```bash
# Вставьте ваш токен бота
BOT_TOKEN=1234567890:ABCDEFGHIJKLMNOPQRSTUVWXYZ

# API данные уже настроены (если есть)
TG_API_ID=your_api_id
TG_API_HASH=your_api_hash

# OpenRouter API ключ уже настроен (если есть)
OPENROUTER_API_KEY=your_openrouter_key
```

### 3. Запуск бота

```bash
cd oracul-bot
python unified_bot.py
```

или

```bash
python bot_self_analysis.py
```
