✅ **Структура проекта:** OK

⚠️ **Конфигурация:** Требует настройки BOT_TOKEN  

## Быстрый запуск

### 1. Настройте BOT_TOKEN

Отредактируйте файл `.env`:
```bash
# Замените YOUR_BOT_TOKEN_HERE на реальный токен
BOT_TOKEN=1234567890:ABCDEFGHIJKLMNOPQRSTUVWXYZ
```

### 2. Запустите бота

```bash
cd oracul-bot
python simple_bot.py
```

## Доступные версии

### 🔹 Simple Bot (Рекомендуется для начала)
```bash
python simple_bot.py
```
- Минимальные зависимости
- Базовый анализ чатов
- Простая статистика

### 🔹 Unified Bot (Полная версия)
```bash
python unified_bot.py
```
- Требует больше зависимостей
- Полный функционал
- AI анализ через OpenRouter

### 🔹 Original Bot (Оригинальная версия)
```bash
python bot_self_analysis.py
```
- Фокус на самоанализе
- Требует DialogSummaryAnalyzer

## Получение BOT_TOKEN

1. Откройте [@BotFather](https://t.me/BotFather) в Telegram
2. Отправьте `/newbot`
3. Введите имя бота: `My Oracul Bot`
4. Введите username: `my_oracul_bot`
5. Скопируйте токен и вставьте в `.env`

## Решение проблем

### Ошибка "BOT_TOKEN не найден"
- Проверьте файл `.env`
- Убедитесь, что токен правильный

### Ошибка "Module not found"
- Установите зависимости: `pip install -r requirements_minimal.txt`

### Ошибка "API_ID не настроен"
- Для полного функционала нужны TG_API_ID и TG_API_HASH
- Получите на https://my.telegram.org
