# 📁 Архив проекта Oracul Bot

Эта папка содержит архивные файлы, которые были перемещены из корня проекта для наведения порядка.

## 📂 Структура архива

### `old_scripts/` - Старые скрипты анализа
- `analyze_*.py` - Скрипты анализа каналов и пользователей
- `collect_*.py` - Скрипты сбора данных
- `find_*.py` - Скрипты поиска
- `get_*.py` - Скрипты получения данных
- `search_*.py` - Скрипты поиска
- `check_*.py` - Скрипты проверки
- `critical_*.py` - Критический анализ
- `simple_*.py` - Простые анализаторы

### `old_tests/` - Тестовые файлы
- `test_*.py` - Различные тесты функциональности
- `kan_model_research.py` - Исследование KAN моделей
- `cleanup_project.py` - Скрипт очистки проекта

### `old_docs/` - Старая документация
- `*_COMPLETE.md` - Отчеты о завершении
- `*_GUIDE.md` - Руководства
- `*_PLAN*.md` - Планы разработки
- `*_REPORT.md` - Отчеты
- `*_SUMMARY.md` - Сводки
- `INTEGRATION_*.md` - Документы интеграции
- `SETUP_*.md` - Инструкции по настройке

### `experiments/` - Экспериментальные файлы
- `ANTIGRAVITY_*.md` - Эксперименты с ANTIGRAVITY
- `uk_impulse_*.md` - Анализ UK Impulse
- `CHAT_*.md` - Анализы чатов
- `*.json` - JSON файлы с данными
- `*.session*` - Файлы сессий Telegram

## 🎯 Цель архивирования

Файлы были перемещены в архив для:
- ✅ Очистки корня проекта
- ✅ Улучшения навигации
- ✅ Сохранения истории разработки
- ✅ Фокуса на активных компонентах

## 🔄 Восстановление файлов

Если нужно восстановить какой-то файл:
```bash
# Переместить файл обратно в корень
Move-Item -Path "archive/old_scripts/analyze_something.py" -Destination "./"
```
