## 🛠️ ИНСТРУКЦИИ ПО ЗАПУСКУ

### 🚀 **Быстрый старт:**
```bash
cd oracul-bot
pip install -r requirements_local.txt
python test_analyzers_only.py  # Проверка
python run_local.py            # Запуск
```

### 🐳 **Docker (рекомендуется):**
```bash
# С GPU поддержкой
docker-compose -f docker-compose-local.yml --profile gpu up -d

# Только CPU
docker-compose -f docker-compose-local.yml --profile cpu up -d
```

### ⚙️ **Настройка .env:**
```env
TELEGRAM_BOT_TOKEN=your-bot-token
DATABASE_URL=postgresql+asyncpg://user:pass@localhost:5432/oracul_db
FORCE_CPU=false  # true для CPU-only режима
```
