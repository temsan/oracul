# 🔮 Oracul Self-Analysis Bot - Руководство

## 🚀 Запуск бота

Бот запущен и работает в фоновом режиме!

```bash
python oracul-bot/bot_self_analysis.py
```

## 📱 Как использовать

### 1. Начало работы
- Найдите бота в Telegram (используйте BOT_TOKEN из .env)
- Отправьте команду `/start`
- Вы увидите главное меню с кнопками

### 2. Анализ по темам

**Доступные темы:**
- 💼 **Работа и карьера** - анализ сообщений о работе, проектах, задачах
- 👥 **Отношения** - анализ общения о друзьях, семье, отношениях
- 🎯 **Цели и планы** - что вы планируете и к чему стремитесь
- 😊 **Эмоции и настроение** - эмоциональные паттерны
- 🧠 **Саморазвитие** - обучение, развитие навыков
- 💭 **Общий анализ** - полный анализ без фильтрации

### 3. Процесс анализа

1. **Выберите тему** в меню "📊 Анализ по темам"
2. **Получите список чатов** командой `/chats`
3. **Отправьте ID чата** для анализа (например: `1637334`)
4. **Дождитесь результата** (обычно 10-30 секунд)

### 4. Что вы получите

✅ **Статистика:**
- Количество сообщений
- Соотношение текст/голос
- Временной промежуток

✅ **Саммари диалога:**
- Основные темы разговоров
- Ключевые моменты
- Общая тональность

✅ **Инсайты:**
- Эмоциональные паттерны
- Динамика общения
- Практические наблюдения

✅ **Голосовой анализ:**
- Длительность голосовых
- Эмоциональные характеристики
- Паттерны речи

## 🎯 Команды

- `/start` - Главное меню
- `/chats` - Список ваших личных чатов
- `/help` - Справка по использованию

## 🔧 Технические детали

### Используемые модели (по приоритету):
1. `nvidia/nemotron-3-nano-30b-a3b:free` - основная (256k контекст)
2. `arcee-ai/trinity-mini:free` - резервная (131k контекст)
3. `openai/gpt-oss-120b:free` - fallback (при доступности)
4. `openai/gpt-oss-20b:free` - fallback (при доступности)

### Возможности:
- ✅ Анализ до 200 сообщений за раз
- ✅ Фильтрация по ключевым словам
- ✅ Мультимодальный анализ (текст + голос)
- ✅ Автоматическое переключение моделей при rate limits
- ✅ Локальная обработка голосовых сообщений

### Конфиденциальность:
- 🔒 Все данные обрабатываются через ваш Telegram аккаунт
- 🔒 Анализ выполняется в реальном времени
- 🔒 Результаты не сохраняются на сервере

## 📊 Пример использования


## 🛠️ Остановка бота

Если нужно остановить бота:
```bash
# Найдите процесс
ps aux | grep bot_self_analysis

# Остановите
kill <PID>
```

Или используйте Ctrl+C в терминале где запущен бот.

## 💡 Советы

1. **Для лучших результатов** анализируйте чаты с 50+ сообщениями
2. **Используйте темы** для фокусированного анализа
3. **Общий анализ** дает полную картину без фильтрации
4. **Голосовые сообщения** анализируются автоматически
5. **Результаты** можно сохранить скриншотом или копированием
