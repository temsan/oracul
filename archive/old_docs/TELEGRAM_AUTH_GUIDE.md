# 🔐 Руководство по авторизации Telegram API

## 📱 **Как работает автоматический сбор данных**

### 🎯 **Что собирает бот:**

1. **Ваши сообщения из чатов** (последние 30 дней, до 100 сообщений)
2. **Ваши посты из каналов** где вы автор/админ (до 50 постов)  
3. **Публичные упоминания** по вашему @username (до 30 сообщений)

### 🔒 **Безопасность:**

- Бот видит **только ваши данные**, не чужие
- Требует **вашего разрешения** через SMS код
- Работает как **обычное Telegram приложение**
- Можно **отозвать доступ** в любой момент командой `/logout`

---

## 🚀 **Пошаговая инструкция**

### 1️⃣ **Получение API ключей Telegram**

1. Перейдите на https://my.telegram.org/apps
2. Войдите с вашим номером телефона
3. Создайте новое приложение:
   - **App title**: `TG Analysis Bot`
   - **Short name**: `tg_analysis`
   - **Platform**: `Desktop`
4. Скопируйте **API ID** и **API Hash**

### 2️⃣ **Настройка .env файла**

```env
# telegram-bot/.env
TELEGRAM_API_ID=your_api_id_here
TELEGRAM_API_HASH=your_api_hash_here
```

### 3️⃣ **Процесс авторизации в боте**

1. **Отправьте** `/analyze` боту
2. **Выберите** анализаторы
3. **Нажмите** "🔍 Собрать из Telegram автоматически"
4. **Нажмите** "🔐 Начать авторизацию"
5. **Введите** номер телефона в формате `+7 900 123 45 67`
6. **Получите SMS** и введите код
7. **При необходимости** введите пароль 2FA
8. **Дождитесь** автоматического сбора данных

## 🛠️ **Управление авторизацией**

### 📱 **Команды:**
- `/logout` - выйти из авторизации и закрыть сессию
- `/collect` - протестировать сбор данных (требует авторизации)
- `/analyze` - запустить новый анализ с авторизацией

### 🔧 **Файлы сессий:**
- Сохраняются в `telegram-bot/sessions/`
- Автоматически создаются для каждого пользователя
- Безопасно удаляются при `/logout`

---

## ⚠️ **Важные моменты**

### 🔐 **Приватность:**
- Бот **НЕ может** читать чужие личные сообщения
- Собирает **только ваши** сообщения и посты
- **НЕ сохраняет** данные после анализа
- Работает **только с вашего разрешения**

### 📱 **Ограничения:**
- Нужен тот же номер телефона, что привязан к Telegram
- Может потребоваться пароль 2FA если включен
- Некоторые приватные чаты могут быть недоступны
- Ограничение по количеству сообщений (100-200 штук)

### 🚫 **Что НЕ может:**
- Читать удаленные сообщения
- Обходить настройки приватности
- Получать доступ к чужим данным
- Работать без вашей авторизации

---

## 🔧 **Устранение проблем**

### ❌ **"Неверный номер телефона"**
- Используйте международный формат: `+7 900 123 45 67`
- Проверьте код страны
- Убедитесь что номер привязан к Telegram

### ❌ **"Неверный код"**
- Проверьте SMS еще раз
- Код обычно 5-6 цифр
- Попробуйте запросить код повторно

### ❌ **"Данные не найдены"**
- Мало активности в последние 30 дней
- Приватные настройки аккаунта
- Используйте ручной ввод сообщений

### ❌ **"Ошибка подключения"**
- Проверьте TELEGRAM_API_ID и TELEGRAM_API_HASH в .env
- Убедитесь что API ключи правильные
- Перезапустите бота

## 🎯 **Рекомендации для лучшего анализа**

### 📝 **Ручной ввод:**
- Минимум 100-200 слов
- Разнообразные темы (работа, хобби, отношения)
- Естественный стиль общения

### 🔍 **Автоматический сбор:**
- Активное общение в чатах
- Посты в каналах или группах
- Публичная активность с @username

### 🎨 **Комбинированный подход:**
1. Используйте автосбор для базы данных
2. Добавьте ручные сообщения для конкретных тем
3. Получите максимально точный анализ
