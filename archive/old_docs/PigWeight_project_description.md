# PigWeight Project - Описание для резюме

## Краткое описание проекта
Проект PigWeight - система для определения веса свиней по фотографиям с использованием компьютерного зрения и машинного обучения.

## Для добавления в резюме:

### Название проекта: 
**Computer Vision система определения веса животных (PigWeight)**

### Описание:
AI-система для автоматического определения веса свиней по фотографиям с использованием компьютерного зрения и машинного обучения.

### Технологии:
- Python
- OpenCV для обработки изображений
- Machine Learning модели для регрессии
- Computer Vision алгоритмы
- Обработка и анализ изображений

### Задача:
Автоматизировать процесс взвешивания животных в сельском хозяйстве через анализ фотографий.

### Результат:
- Система определения веса по визуальным признакам
- Автоматизация процесса взвешивания
- Применение CV в агротехе

### Роль:
ML Engineer / Computer Vision Developer
