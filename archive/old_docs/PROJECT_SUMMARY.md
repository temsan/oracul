## 🔧 API АРХИТЕКТУРА

### Основные Endpoints

```
Authentication:
POST /api/v1/auth/telegram          # Авторизация в Telegram
POST /api/v1/auth/telegram/code     # Подтверждение кода
GET  /api/v1/auth/telegram/status   # Статус авторизации

Data Collection:
POST /api/v1/collect/start          # Запуск сбора данных
GET  /api/v1/collect/status/{id}    # Статус сбора
GET  /api/v1/collect/results/{id}   # Результаты сбора

Analysis:
GET  /api/v1/analyze/types          # Доступные типы анализа
POST /api/v1/analyze/start          # Запуск анализа
GET  /api/v1/analyze/status/{id}    # Статус анализа
GET  /api/v1/analyze/results/{id}   # Результаты анализа

Reports:
GET  /api/v1/report/templates       # Шаблоны отчетов
POST /api/v1/report/generate        # Генерация отчета
GET  /api/v1/report/download/{id}   # Скачивание отчета
```
