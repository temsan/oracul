# Обзор продукта

## Oracul Bot - Telegram бот для самоанализа

Персональный AI-бот для глубокого анализа собственных Telegram диалогов и психологического самопознания. Система обеспечивает этичное профилирование личности через мультимодальный анализ текста, голоса и изображений.

### Основные возможности

- **💬 Самоанализ диалогов**: Анализ личных чатов с различной глубиной (50-200 сообщений)
- **🧠 Психологический анализ**: Big Five, архетипы Юнга, стили привязанности, путь героя
- **🎤 Анализ голосовых**: Уникальный анализ эмоционального состояния через голосовые сообщения
- **📊 Каналы и группы**: Анализ влияния подписок на личность
- **💼 Карьерный анализ**: Поиск работы и карьерные рекомендации
- **🔮 Анализ ситуации**: Комплексный анализ текущего жизненного этапа

### Целевые сценарии использования

- **Личностное развитие**: Самоанализ и психологические инсайты
- **Самопознание**: Понимание собственных коммуникационных паттернов
- **Отношения**: Анализ динамики общения с близкими людьми
- **Карьерное развитие**: Оптимизация поиска работы через самоанализ
- **Эмоциональный интеллект**: Развитие через анализ голосовых сообщений

### Уникальное ценностное предложение

Персональный AI-помощник для самопознания, который сочетает этичный анализ с психологическими фреймворками для получения практических инсайтов из собственных паттернов цифровой коммуникации. Особая сила в анализе голосовых сообщений - уникальной возможности для понимания эмоционального состояния.

## Связанный проект: Oracul Analysis Engine

Отдельная платформа для анализа каналов, пользователей и карьерных возможностей, вынесенная в независимый проект `oracul-analysis-engine/` для бизнес-аналитики и исследований.
