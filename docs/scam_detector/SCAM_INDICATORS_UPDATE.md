## 🎥 Новый индикатор: Запрос на демонстрацию экрана

### Описание

Мошенники просят расшарить (показать) экран во время видеозвонка под различными предлогами.

### Вес: 8 баллов (высокий риск)

### Паттерны:

- "расшарить экран"
- "показать экран"
- "демонстрация экрана"
- "трансляция экрана"
- "screen sharing"
- "видеозвонок с экраном"
- "звонок с экраном"
- "включить запись"

### Почему это опасно:

1. **Видят ваши пароли** при вводе
2. **Видят коды из SMS** на экране
3. **Видят данные карт** в банке
4. **Видят личную информацию** в документах
5. **Могут записать** все действия

### Типичные предлоги:

- "Для записи собеседования"
- "Чтобы проверить навыки"
- "Для заполнения анкеты"
- "Чтобы показать рабочую систему"
- "Требование компании"

### Реальная цель:

Украсть доступ к вашим аккаунтам:
- Банк → списать деньги
- Госуслуги → оформить кредиты
- Почта → получить доступ к другим сервисам
- Криптокошельки → украсть криптовалюту

## 📹 Новый индикатор: Упоминание видеозвонка

### Описание

Само по себе упоминание видеозвонка не подозрительно, но в комбинации с другими признаками повышает риск.

### Вес: 1 балл (низкий риск)

### Паттерны:

- "видеозвонок"
- "видео звонок"
- "видеосвязь"
- "видео собеседование"
- "zoom собеседование"
- "skype собеседование"

### Когда это нормально:

✅ Легитимные компании используют видеозвонки для:
- Знакомства с командой
- Обсуждения проекта
- Оценки коммуникативных навыков
- Финального этапа собеседования

### Когда это подозрительно:

❌ В комбинации с:
- Запросом на расшаривание экрана
- Отсутствием названия компании
- Новым аккаунтом отправителя
- Завышенной зарплатой
- Контактом только через ЛС

## 🧪 Примеры работы

### Пример 1: Схема с расшариванием

**Текст:**
```
Требуется редактор. Зарплата 68 тр.
Собеседование по видеосвязи с расшариванием экрана.
Контакт: @hr_manager
```

**Результат:**
- 🟠 СРЕДНИЙ РИСК - 14 баллов
- Найдено:
  - 💸 Завышенная зарплата (+3)
  - 📱 Только ЛС (+2)
  - 🎥 Расшаривание экрана (+8)
  - 📹 Видеозвонок (+1)

### Пример 2: Легитимная вакансия

**Текст:**
```
Яндекс.Дзен ищет редактора.
Зарплата: 45-55K
Собеседование: видеозвонок с командой.
Контакты: hr@yandex-team.ru
```

**Результат:**
- 🟢 МИНИМАЛЬНЫЙ РИСК - 3 балла
- Найдено:
  - 📱 Только ЛС (+2) [ложное срабатывание на email]
  - 📹 Видеозвонок (+1)

### Пример 3: Критическая схема

**Текст:**
```
SMM-менеджер. 80K + бонусы.
Собеседование по Zoom. Расшарить экран и показать 
навыки. Также открыть банк для реквизитов.
Пишите: @scammer
```

**Результат:**
- 🟠 СРЕДНИЙ РИСК - 10 баллов
- Найдено:
  - 📱 Только ЛС (+2)
  - 🎥 Расшаривание экрана (+8)

## 🎯 Как использовать

### Автоматическая проверка:

```bash
python scripts/career_job_search/scam_detection/check_job_scam.py
```

Детектор автоматически выявит запрос на расшаривание экрана.

### Программное использование:

```python
from analyzers.job_scam_analyzer import JobScamAnalyzer

text = """
Вакансия редактора. Собеседование с расшариванием экрана.
"""

analyzer = JobScamAnalyzer()
result = analyzer.analyze(text)

# Проверяем наличие индикатора
has_screen_sharing = any(
    'демонстрацию экрана' in ind['description'].lower()
    for ind in result['indicators']
)

if has_screen_sharing:
    print("⚠️ ОПАСНО: Запрос на расшаривание экрана!")
```

## 🛡️ Рекомендации

### Если видите запрос на расшаривание:

1. ❌ **НЕ откликайтесь** на такую вакансию
2. 🚫 **Заблокируйте** контакт
3. 📢 **Предупредите** других
4. 📝 **Сообщите** о мошенничестве

### Легитимные альтернативы:

✅ Компании могут попросить:
- Портфолио по ссылке
- Примеры работ файлами
- Тестовое задание отдельно
- Презентацию без расшаривания

❌ Компании НЕ должны просить:
- Расшарить экран
- Открыть банк/госуслуги
- Ввести пароли
- Зачитать коды
