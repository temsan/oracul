### ✅ Функции:

1. **Анализ текста вакансии**
   - 9 типов индикаторов мошенничества
   - Система оценки 0-30 баллов
   - 4 уровня риска

2. **Проверка отправителя**
   - Анализ ID аккаунта
   - Определение возраста аккаунта
   - Проверка на скам-метки Telegram

3. **Поиск похожих вакансий**
   - Сканирование каналов
   - Поиск по контактам
   - Выявление сетей мошенников

4. **Детальные отчеты**
   - JSON для программной обработки
   - Markdown для чтения
   - Рекомендации по действиям

5. **Интеграция с ботом**
   - Команда `/checkjob` в Telegram
   - Автоматическая проверка пересланных сообщений
   - Форматированные отчеты

## 🛡️ Как использовать

### Сценарий 1: Получили вакансию в ЛС

```bash
python scripts/career_job_search/scam_detection/check_job_scam.py
```

Вставьте текст → Получите вердикт.

### Сценарий 2: Нашли вакансию в канале

```bash
# Измените ссылку в скрипте
python scripts/career_job_search/scam_detection/check_specific_job.py
```

Получите анализ + похожие вакансии.

### Сценарий 3: Подозреваете сеть мошенников

```bash
# Добавьте контакты в SUSPICIOUS_CONTACTS
python scripts/career_job_search/scam_detection/find_scam_network.py
```

Найдите все вакансии от этих контактов.

### Сценарий 4: Используете Telegram бота

1. Интегрируйте обработчики в `oracul-bot/main.py`
2. Используйте `/checkjob` в Telegram
3. Или просто перешлите вакансию боту
