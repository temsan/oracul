# 🚨 Детектор мошеннических вакансий

Автоматическая система проверки вакансий на признаки мошенничества в Telegram.

## 🎯 Что проверяет система

### Критические признаки (10+ баллов):
- **Запросы на оплату** - просьбы оплатить обучение, регистрацию, материалы
- **Фейковые гарантии** - "система безопасных сделок", "гарант оплаты"

### Высокий риск (5-8 баллов):
- **Новый аккаунт** - ID > 5 млрд (созданы недавно)
- **Завышенная зарплата** - для редактора > 60K необычно
- **Размытые формулировки** - нет названия компании, общие фразы

### Средний риск (2-3 балла):
- **Контакт через ЛС** - только через личку, без официальных каналов
- **Подозрительные требования** - "только из РФ", "специальная карта"
- **Слишком хорошо** - "без опыта", "гарантированный доход"
- **Общие задачи** - "различные типы текстов" без конкретики

## 🚀 Использование

### Вариант 1: Быстрая проверка (интерактивно)
```bash
python check_job_scam.py
```

Вставьте текст вакансии, при необходимости укажите ID отправителя.

### Вариант 2: Программная проверка
```python
from scripts.career_job_search.detect_job_scam import check_job_posting
from telethon import TelegramClient

client = TelegramClient('session', api_id, api_hash)
await client.start()

analysis = await check_job_posting(
    client=client,
    message_text="текст вакансии",
    sender_id=8345148786  # опционально
)

print(analysis['verdict']['risk_level'])
```

### Вариант 3: Полный анализ с отчетом

## 🛡️ Как защититься от мошенников

### ❌ НИКОГДА не делайте:
1. Не переводите деньги (за обучение, регистрацию, материалы)
2. Не скачивайте подозрительные приложения
3. Не показывайте экран по видеосвязи с кодами доступа
4. Не вводите данные карты на незнакомых сайтах
5. Не заходите в чужой Apple ID

### ✅ ВСЕГДА проверяйте:
1. Название компании (поиск в Google, отзывы)
2. Официальный сайт компании
3. Наличие вакансии на сайте компании
4. Отзывы о компании на специализированных ресурсах
5. Возраст домена сайта (whois-service.ru)

### 🔍 Красные флаги:
- Слишком высокая зарплата для позиции
- Нет названия компании
- Контакт только через ЛС
- Просят оплатить что-либо
- "Гарантированный доход"
- Новый аккаунт отправителя
- Общие формулировки без конкретики
