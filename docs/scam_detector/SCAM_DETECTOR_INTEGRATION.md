# 🚨 Интеграция детектора мошеннических вакансий

## 📦 Что создано

### 1. Standalone скрипты
- `scripts/career_job_search/detect_job_scam.py` - Полный анализатор с Telegram API
- `check_job_scam.py` - Быстрая интерактивная проверка

### 2. Интеграция с Oracul Bot
- `oracul-bot/analyzers/job_scam_analyzer.py` - Анализатор (без зависимостей от Telegram)
- `oracul-bot/bot/job_scam_handler.py` - Обработчики команд для бота

### 3. Документация
- `analysis/career_development/SCAM_DETECTOR_GUIDE.md` - Руководство пользователя

## 🚀 Использование

### Вариант 1: Standalone проверка

```bash
# Интерактивная проверка
python check_job_scam.py

# Полный анализ с примером
python scripts/career_job_search/detect_job_scam.py
```

### Вариант 2: Через Oracul Bot

#### Шаг 1: Добавить обработчики в main.py

Откройте `oracul-bot/main.py` и добавьте:

```python
from bot.job_scam_handler import register_job_scam_handlers

# В функции main() после создания application:
def main():
    application = Application.builder().token(BOT_TOKEN).build()
    
    # ... существующие обработчики ...
    
    # Добавляем обработчики проверки вакансий
    register_job_scam_handlers(application)
    
    # ... остальной код ...
```

#### Шаг 2: Обновить список команд

В `oracul-bot/bot/handlers.py` добавьте в команду `/start`:

```python
*Команды:*
/start - Начало работы
/analyze - Анализ текста
/checkjob - 🚨 Проверка вакансии на мошенничество
/help - Помощь
```

#### Шаг 3: Перезапустить бота

```bash
cd oracul-bot
python main.py
```

### Вариант 3: Программное использование

```python
from oracul-bot.analyzers.job_scam_analyzer import JobScamAnalyzer

analyzer = JobScamAnalyzer()

# Анализ текста
result = analyzer.analyze(
    text="текст вакансии",
    sender_id=8345148786  # опционально
)

# Форматированный отчет
report = analyzer.format_report(result)
print(report)

# Быстрая проверка
verdict = analyzer.quick_check("текст вакансии")
print(verdict)
```

## 📱 Команды бота

### `/checkjob` - Проверка вакансии

**Использование:**
1. Отправьте команду `/checkjob`
2. Бот попросит прислать текст вакансии
3. Отправьте текст или перешлите сообщение
4. Получите детальный анализ

**Пример:**
```
Пользователь: /checkjob
Бот: Отправьте мне текст вакансии...

Пользователь: [пересылает сообщение с вакансией]
Бот: 🚨 ПРОВЕРКА ВАКАНСИИ НА МОШЕННИЧЕСТВО
     
     Уровень риска: 🔴 ВЫСОКИЙ РИСК
     Баллов: 17
     
     💡 Рекомендация:
     НЕ ОТКЛИКАЙТЕСЬ! Множественные признаки мошенничества.
     
     ⚠️ Найдено признаков: 5
     • 🆕 Новый аккаунт (+5)
     • 💸 Завышенная зарплата (+3)
     • 📱 Контакт только через ЛС (+2)
     ...
```

### Автоматическая проверка

Бот автоматически проверяет пересланные сообщения, если они:
- Содержат ключевые слова вакансий
- Имеют признаки риска (≥4 балла)

**Пример:**
```
Пользователь: [пересылает сообщение с вакансией]
Бот: ⚠️ Автоматическая проверка вакансии
     
     🔴 ВЫСОКИЙ РИСК (17 баллов)
     НЕ ОТКЛИКАЙТЕСЬ! Множественные признаки мошенничества.
     
     Для подробного анализа используйте /checkjob
```

## 🎯 Признаки мошенничества

### Критические (10+ баллов):
- 💰 Запросы на оплату обучения/регистрации
- 🛡️ Фейковые "системы безопасных сделок"

### Высокий риск (5-8 баллов):
- 🆕 Новый аккаунт отправителя
- 💸 Завышенная зарплата для позиции

### Средний риск (2-3 балла):
- 📱 Контакт только через ЛС
- 🏢 Нет названия компании
- ⚠️ Подозрительные требования
- ✨ "Слишком хорошо, чтобы быть правдой"
- 📋 Общие задачи без конкретики

## 📊 Уровни риска

| Баллы | Уровень | Действие |
|-------|---------|----------|
| 15+ | 🔴 ВЫСОКИЙ | НЕ откликайтесь |
| 8-14 | 🟠 СРЕДНИЙ | Будьте осторожны |
| 4-7 | 🟡 НИЗКИЙ | Требует проверки |
| 0-3 | 🟢 МИНИМАЛЬНЫЙ | Можно откликаться |
