# 🎥 Схема мошенничества с расшариванием экрана

**Уровень опасности:** 🔴 КРИТИЧЕСКИЙ  
**Распространенность:** Очень высокая  
**Потенциальный ущерб:** Полная кража аккаунтов и денег

---

## 🚨 Описание схемы

Мошенники просят расшарить (показать) экран во время видеозвонка под предлогом:
- "Для включения записи собеседования"
- "Чтобы проверить ваши навыки работы с программами"
- "Для заполнения анкеты на экране"
- "Чтобы показать рабочую систему"

### Что происходит на самом деле:

1. **Вы расшариваете экран** на Zoom/Skype/Telegram
2. **Мошенник просит открыть** банк/госуслуги/почту
3. **Вы вводите пароль** - мошенник видит его на экране
4. **Приходит SMS с кодом** - мошенник просит "зачитать для проверки"
5. **Аккаунт украден** - деньги списаны, доступ потерян

---

## 🎯 Ваш случай

Вы упомянули, что вас пытались развести на расшаривание экрана при звонке.

### Типичный сценарий:

```
Мошенник: "Здравствуйте! Мы рассмотрели вашу заявку на вакансию 
           редактора. Давайте проведем собеседование по видеосвязи."

Вы: "Хорошо"

Мошенник: "Для записи собеседования нужно расшарить экран. 
           Это требование нашей компании для безопасности."

Вы: [расшариваете экран]

Мошенник: "Отлично! Теперь откройте, пожалуйста, ваш банковский 
           аккаунт, чтобы мы могли проверить реквизиты для зарплаты."

Вы: [открываете банк, вводите пароль]

Мошенник: [ВИДИТ ВАШ ПАРОЛЬ НА ЭКРАНЕ]

Мошенник: "Вам пришло SMS? Зачитайте код для подтверждения."

Вы: [зачитываете код]

Мошенник: [ВХОДИТ В ВАШ АККАУНТ, ПЕРЕВОДИТ ДЕНЬГИ]

Результат: Деньги украдены, аккаунт взломан
```

---

## ⚠️ Признаки этой схемы

### 🔴 Критические (немедленно прекращайте общение):

1. **Просят расшарить экран** на любом этапе
2. **Просят открыть банк/госуслуги** во время звонка
3. **Просят ввести пароли** при включенной демонстрации
4. **Просят зачитать коды из SMS**
5. **Давят на срочность** ("быстро, пока не закрылась сессия")

### 🟠 Подозрительные:

1. Видеозвонок на **первом** этапе (до реального собеседования)
2. Требование записи собеседования
3. Просьба показать "навыки работы с программами"
4. Заполнение анкеты на вашем экране
5. Проверка "технических возможностей"

---

## 🛡️ Как защититься

### ❌ НИКОГДА не делайте:

1. **НЕ расшаривайте экран** на собеседованиях
2. **НЕ открывайте банк/госуслуги** во время звонков
3. **НЕ вводите пароли** при включенной демонстрации
4. **НЕ зачитывайте коды** из SMS никому
5. **НЕ показывайте личные данные** на экране

### ✅ Правильные действия:

1. **Откажитесь от расшаривания:**
   ```
   "Извините, я не расшариваю экран на собеседованиях. 
   Это вопрос безопасности моих данных."
   ```

2. **Предложите альтернативу:**
   ```
   "Могу показать портфолио через ссылку или 
   выслать примеры работ отдельными файлами."
   ```

3. **Проверьте компанию:**
   - Найдите официальный сайт
   - Позвоните в офис
   - Проверьте отзывы

4. **Если настаивают - это скам:**
   ```
   Легитимные компании НИКОГДА не требуют 
   расшаривания экрана на собеседовании.
   ```

## 🎓 Легитимные vs Мошеннические собеседования

### ✅ Легитимное собеседование:

```
1. Первый этап: Телефонный звонок или текстовая переписка
2. Второй этап: Видеозвонок БЕЗ расшаривания экрана
3. Третий этап: Тестовое задание (отдельно, без звонка)
4. Четвертый этап: Встреча в офисе или финальный звонок

НЕ ПРОСЯТ:
❌ Расшарить экран
❌ Открыть банк
❌ Ввести пароли
❌ Зачитать коды
❌ Показать личные данные
```

### ❌ Мошенническое "собеседование":

```
1. Сразу предлагают видеозвонок
2. Просят расшарить экран "для записи"
3. Просят открыть банк/госуслуги
4. Просят ввести пароли
5. Просят зачитать коды из SMS

КРАСНЫЕ ФЛАГИ:
🚩 Срочность ("быстро, пока не закрылась сессия")
🚩 Давление ("это обязательное требование")
🚩 Странные просьбы ("откройте банк для реквизитов")
🚩 Нет официального сайта компании
🚩 Новый аккаунт отправителя
```

## 🔍 Как проверить компанию

### Перед видеозвонком:

1. **Найдите официальный сайт:**
   - Google: "название компании официальный сайт"
   - Проверьте возраст домена (whois-service.ru)
   - Должен быть старше 1 года

2. **Проверьте вакансию на сайте:**
   - Есть ли эта вакансия на официальном сайте?
   - Совпадают ли условия?
   - Указаны ли официальные контакты?

3. **Позвоните в офис:**
   - Найдите телефон на официальном сайте
   - Спросите про вакансию
   - Уточните процесс собеседования

4. **Проверьте отзывы:**
   - Хабр Карьера
   - VC.ru
   - Google отзывы
   - Telegram каналы с отзывами

### Во время звонка:

1. **Спросите про компанию:**
   - "Расскажите о вашей компании"
   - "Какие проекты у вас есть?"
   - "Где находится офис?"

2. **Уточните процесс:**
   - "Какие этапы собеседования?"
   - "Зачем нужна демонстрация экрана?"
   - "Можно ли без расшаривания?"

3. **Если что-то не так:**
   - Прекратите звонок
   - Заблокируйте контакт
   - Сообщите о мошенничестве

---

## 📱 Что делать, если уже расшарили

### Если показали пароли:

1. **НЕМЕДЛЕННО смените пароли:**
   - Банк
   - Госуслуги
   - Почта
   - Все важные аккаунты

2. **Проверьте операции:**
   - Банковские транзакции
   - Активность в аккаунтах
   - Подозрительные входы

3. **Заблокируйте карты:**
   - Позвоните в банк
   - Заблокируйте все карты
   - Закажите новые

4. **Включите 2FA:**
   - Двухфакторная аутентификация
   - На всех важных сервисах
   - Используйте приложения (не SMS)

### Если зачитали коды:

1. **Срочно звоните в банк:**
   - Сообщите о мошенничестве
   - Заблокируйте все операции
   - Оспорьте транзакции

2. **Заявление в полицию:**
   - Напишите заявление
   - Приложите скриншоты переписки
   - Укажите все детали

3. **Сообщите в Telegram:**
   - Report → Spam
   - Опишите ситуацию
   - Заблокируйте мошенника
