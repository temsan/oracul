```
oracul-analysis-engine/
```

## 🗑️ Что было удалено/упрощено

### ❌ Удален избыточный бэкенд
- **Удалено**: `oracul-analysis-engine/backend/` (FastAPI, PostgreSQL, Redis)
- **Причина**: Избыточность для простых скриптов анализа
- **Результат**: Упрощенная архитектура без лишних зависимостей

### 📦 Перенесены конфигурации
- **Перенесено**: `.env` → `oracul-analysis-engine/.env`
- **Перенесено**: `oracul.session` → `oracul-analysis-engine/oracul.session`
- **Результат**: Каждый проект имеет свою конфигурацию

### 🐳 Упрощен Docker
- **Было**: PostgreSQL + Redis + Backend + CLIProxy
- **Стало**: CLIProxy + Jupyter (опционально)
- **Результат**: Минимальная инфраструктура
