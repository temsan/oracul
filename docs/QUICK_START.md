## 🚀 НЕМЕДЛЕННЫЕ ДЕЙСТВИЯ

### 1️⃣ **ЗАПУСК ORACUL BOT:**
```bash
# Основной объединенный бот
cd oracul-bot
python unified_bot.py

# Или оригинальный бот самоанализа
python bot_self_analysis.py
```

### 2️⃣ **АНАЛИЗ ДАННЫХ (ОТДЕЛЬНЫЙ ПРОЕКТ):**
```bash
# Перейти в проект анализов
cd oracul-analysis-engine

# Запустить анализ
python run_analysis.py

# Или через Docker
docker-compose up -d
```

### 3️⃣ **НАСТРОЙКА:**
```bash
# Настроить переменные окружения
cd oracul-bot
cp .env.example .env
# Отредактируйте .env с вашими API ключами
```
