# 🔮 Oracul Bot - Telegram Analysis Platform

Платформа для анализа Telegram диалогов с использованием ИИ и локальных моделей.

## 🚀 Основные компоненты

### 📱 Боты
- **`oracul-bot/`** - Основной бот для самоанализа диалогов
- **`telegram-bot/`** - Расширенный бот с множественным анализом

### 🧠 Анализ
- **`backend/`** - Backend API для анализа
- **`analysis/`** - Результаты и отчеты анализа

### 🛠️ Инфраструктура  
- **`CLIProxyAPIPlus/`** - Прокси для API доступа
- **`scripts/`** - Утилиты и скрипты
- **`docs/`** - Документация

### 📁 Архив
- **`archive/`** - Архивные файлы и старые эксперименты

## ⚡ Быстрый старт

### Запуск основного бота
```bash
cd oracul-bot
python bot_self_analysis.py
```

### Запуск расширенного бота
```bash
cd telegram-bot  
python bot.py
```

## 🔧 Настройка

1. Скопируйте `.env.example` в `.env`
2. Заполните необходимые API ключи
3. Установите зависимости: `pip install -r requirements.txt`

## 📊 Возможности

- 🎤 Анализ голосовых сообщений с CUDA

## 🔮 Oracul Bot Features

- ✅ Фильтрация ботов из списка чатов
