## 🎯 Что такое Oracul Engine?

**Oracul Engine** - это комплексная система анализа данных из Telegram, которая превратилась из простого консолидатора скриптов в полноценную аналитическую платформу.

### 🚀 Ключевые возможности:
- 📊 **20 модулей** для различных типов анализа
- 🎤 **Анализ голосовых сообщений** (уникальная функция!)
- 📺 **Глубокий анализ каналов** и мониторинг трендов
- 🧠 **Психологические инсайты** и рекомендации
- 💼 **Карьерная аналитика** и поиск работы
- 📈 **Профессиональные отчеты** и дашборды

## 📦 Полный список модулей (20 штук)

### 📡 DATA_COLLECTION (3 модуля)
- **collect_user_data** - Сбор сообщений и данных пользователя

- **collect_voice_messages** - Сбор голосовых сообщений пользователя

- **analyze_channel_content** - Анализ контента (тренды/аудитория/монетизация)

### 💼 CAREER_SEARCH (4 модуля)

- **find_networking_contacts** - Поиск контактов для нетворкинга

### 🧠 PSYCHOLOGICAL_ANALYSIS (5 модулей)
- **analyze_toxic_relationship** - Анализ токсичных отношений
- **analyze_attachment_style** - Определение стиля привязанности
- **analyze_shadow_work** - Работа с теневыми аспектами
- **generate_healing_plan** - План психологического исцеления
- **analyze_current_situation** - Анализ текущей жизненной ситуации

## 🎤 Уникальная функция: Анализ голосовых сообщений

### Что анализируется:
- **Временные паттерны** - когда и как часто отправляются голосовые
- **Длительность сообщений** - короткие/средние/длинные
- **Эмоциональное состояние** - по частоте и времени отправки
- **Стиль общения** - тип коммуникатора
- **Жизненная ситуация** - текущее состояние и рекомендации

### Пример использования:
```python
# Сбор голосовых за неделю
Task(
    task_type=TaskType.DATA_COLLECTION,
    name="collect_voice_messages",
    params={"user_id": 123456, "days_back": 7}
)

# Анализ эмоционального состояния
Task(
    task_type=TaskType.USER_ANALYSIS,
    name="analyze_voice_patterns",
    params={"analysis_focus": "emotional_state"}
)
```
